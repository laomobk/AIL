# AIL 1.2 Alpha 2


AIL 是一门面向对象的编程语言。具有许多高级易用的数据结构以及语法简洁，功能强大的特点。

## Hello World

#### Hello World with one statement
```
print('Hello World')
```
..or..
```
console.writeln('Hello World!')
```

#### Hello World in function
```
func helloWorld() {
    print 'Hello World!'
}

helloWorld()
```

#### Hello World in anonymous function
```
(func () {
    print 'Hello World!'
})()
```

#### Hello World in class
```
class Hello {
    func helloWorld(self) {
        print 'Hello World!'
    }
}

Hello().helloWorld()
```
